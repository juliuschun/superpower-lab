# 여덟 가지 슈퍼파워 — 내적 동기, 직업, 그리고 성공의 원인

> 사람은 자기 내적 동기에 대해서만 일관적이다.
> 같은 상황에서 다른 행동을 하는 이유는, 작동하는 동기가 다르기 때문이다.

---

## 이 문서는 무엇인가

모든 사람에게는 **슈퍼파워**가 있다. 그것은 누가 시키지 않아도 하는 행동, 쉬어야 할 때도 멈추지 못하는 몰입, 화가 날 때조차 돌아가는 관심사 — 그 밑바닥에 있는 **내적 동기**다.

이 문서는:
1. 11개 주요 심리학·강점 프레임워크를 교차 분석하여
2. 사람의 근원적 내적 동기를 **8가지**로 분류하고
3. 각 슈퍼파워가 만들어내는 **직업 경로**와 **성공 원인**을 분석한다

핵심 전제: **"잘하는 것"이 아니라 "누가 시키지 않아도 하는 것"**에 초점을 맞춘다.

| 프레임워크 | 측정 대상 | 카테고리 수 |
|-----------|----------|------------|
| Reiss 16 Basic Desires | 무엇을 원하는가 (욕구) | 16 |

| Gallup CliftonStrengths | 무엇을 잘하는가 (능력) | 34 |

이 11개를 겹쳐 놓으면, 표면 아래의 근원적 동기는 **8갈래**로 수렴한다.

## 여덟 가지 슈퍼파워 전체 지도

```
┌──────────────────────────────────────────────────────────┐
│                   8가지 슈퍼파워                          │
├──────────┬───────────────────────────────────────────────┤
│ 1. 탐구   │ 세상을 이해하고 싶다                          │
│ 2. 창조   │ 없던 것을 만들어야 직성이 풀린다              │
│ 3. 설득   │ 내 논리로 누군가의 생각을 바꾸고 싶다         │
│ 4. 관계   │ 눈앞의 사람을 실망시키지 않겠다               │
│ 5. 성취   │ 목표를 세우고 넘는 쾌감이 전부다              │
│ 6. 수호   │ 이것이 무너지지 않게 지키겠다                 │
│ 7. 의미   │ 세상에 어떤 의미가 있는지 알고 싶다           │
│ 8. 감별   │ 이 사람의 진짜 재능이 뭔지 알고 싶다          │
└──────────┴───────────────────────────────────────────────┘
```

## 1. 탐구 (Explorer)

### 핵심 질문
**"이건 왜 이런 거지?"**

### 무의식적 행동 패턴
- 쉬는 시간에도 논문·책·다큐를 본다
- 모르는 것을 그냥 넘기지 못한다
- 대화 중 "근데 그게 왜 그런 거야?"를 입버릇처럼 한다
- 하나를 알면 그 옆의 것도 알아야 잠이 온다

### 관련 프레임워크
- Reiss: **Curiosity** (호기심)
- CliftonStrengths: **Learner, Input, Intellection**
- VIA: **Love of Learning, Curiosity**
- Working Genius: **Wonder**
- Kolbe: Fact Finder 高

### 이 슈퍼파워가 만드는 직업들

| 직업 | 왜 이 직업인가 | 탐구가 작동하는 방식 |
|------|---------------|---------------------|
| 연구원/과학자 | 가장 직접적인 탐구의 직업화 | 하나의 질문을 10년 파고든다 |
| 전략 컨설턴트 | 산업을 이해하는 것 자체가 보수 | 매 프로젝트가 새로운 세계 |
| 저널리스트/논픽션 작가 | 취재가 곧 탐구 | "왜?"를 묻는 것이 업무 |
| 투자 리서처/애널리스트 | 기업·산업 분석이 탐구의 형태 | 숫자 뒤의 이유를 찾는다 |
| 데이터 사이언티스트 | 패턴을 발견하는 희열 | 데이터 속 "왜?"를 푼다 |
| 교수/강사 | 알게 된 것을 전달하는 쾌감 | 연구 + 설명의 결합 |

### 성공 원인

> **탐구형이 성공하려면, 호기심을 "출력"으로 바꾸는 파이프라인이 필요하다.**

탐구형의 함정은 **영원한 인풋**이다. 알면 알수록 더 알고 싶고, 그래서 결과물이 안 나온다. 성공하는 탐구형은:

1. **마감을 자기에게 건다** — "여기까지 알면 쓴다/말한다/만든다"는 기준선을 정한다
2. **설명할 수 있으면 이해한 것이다** — 누군가에게 설명하는 과정이 탐구의 완성. 그래서 강의·글쓰기·팟캐스트가 좋은 출력 채널
3. **T자형으로 판다** — 넓게 아는 것(가로) + 하나를 깊이 파는 것(세로). 넓기만 하면 잡학, 깊기만 하면 고립

### 흔한 오인
- **"머리가 좋다"**고 자신을 정의 → 사실은 호기심이 슈퍼파워. 머리는 결과이지 원인이 아니다
- **"학벌이 무기"**라고 생각 → 학벌은 호기심이 만든 부산물일 뿐

### 멀티커리어 패턴
탐구형은 자연스럽게 **여러 분야를 넘나든다**. 한 분야를 충분히 이해하면 다음 분야로 옮긴다. 그래서 "전직이 많은 사람" 중에 탐구형이 많다. 그러나 이것은 변덕이 아니라, 탐구의 대상이 바뀐 것이다.

## 2. 창조 (Creator)

### 핵심 질문
**"이걸 이렇게 만들면 어떨까?"**

### 무의식적 행동 패턴
- 기존 것을 그대로 쓰지 못한다. 항상 뭔가를 바꾸거나 새로 짓는다
- "이거 이런 식으로 하면 되겠는데" 가 입버릇
- 완성된 것보다 시작하는 순간이 가장 흥분된다
- 남의 프레임워크보다 자기 프레임워크를 만들고 싶다

### 관련 프레임워크
- Working Genius: **Invention, Wonder**
- Kolbe: **Quick Start 高, Follow Thru 低**
- Wealth Dynamics: **Creator**
- CliftonStrengths: **Ideation, Strategic, Futuristic**
- VIA: **Creativity**

### 이 슈퍼파워가 만드는 직업들

| 직업 | 왜 이 직업인가 | 창조가 작동하는 방식 |
|------|---------------|---------------------|
| 창업가 | 없던 것을 세상에 만든다 | 0→1이 본업 |
| 제품 디자이너/PM | 제품의 형태를 결정한다 | "이렇게 만들면?"이 매일의 질문 |
| 건축가/공간 디자이너 | 물리적 창조 | 빈 땅에서 구조를 만든다 |
| 작곡가/예술가 | 순수한 형태의 창조 | 없던 것을 존재하게 한다 |
| 콘텐츠 크리에이터 | 매일 새로운 것을 만든다 | 콘텐츠 = 일상적 창조 행위 |
| 요리사/셰프 | 재료를 조합해 새 경험을 만든다 | 레시피 = 창조의 언어 |

### 성공 원인

> **창조형이 성공하려면, "시작하는 힘"과 "끝내는 힘"을 분리해야 한다.**

창조형의 함정은 **영원한 시작**이다. 새 프로젝트를 시작하는 흥분은 중독적이고, 80% 완성된 프로젝트를 마무리하는 건 고역이다. 성공하는 창조형은:

1. **파트너를 찾는다** — 시작은 내가, 마무리는 파트너가. Working Genius에서 Invention(창조형) + Tenacity(집념형)의 조합이 가장 강력
2. **MVP를 습관화한다** — 100% 완성 전에 세상에 내놓는 연습. 60%짜리를 빨리 내놓고 피드백으로 완성
3. **포트폴리오로 증명한다** — 창조형의 이력서는 쓸모없다. "이걸 만들었습니다"가 유일한 증명

### 흔한 오인
- **"아이디어가 많다"**고 자신을 정의 → 사실은 아이디어를 현실로 만들려는 충동이 핵심. 아이디어만 있고 안 만드는 사람은 탐구형에 가깝다
- **"끈기가 없다"**고 자책 → 끈기의 문제가 아니라 관심의 대상이 바뀐 것. 새 것에 대한 흥분은 끈기의 부재가 아니라 창조 본능의 발현

### 멀티커리어 패턴
창조형은 **연쇄 창업가**가 되거나, **여러 사이드 프로젝트**를 동시에 돌린다. 하나의 직업에 평생 머무는 경우는 드물다. 이것은 산만함이 아니라, 창조 대상의 자연스러운 전환이다.

## 3. 설득 (Persuader)

### 핵심 질문
**"내 논리로 저 사람의 생각을 바꿀 수 있을까?"**

### 무의식적 행동 패턴
- 어떤 모임이든 "설득 실험의 장"으로 삼는다
- 논리를 가다듬는 데 중독적으로 시간을 쓴다
- 자기 주장에 대한 반론을 미리 준비한다
- 누군가가 "네 말이 맞아"라고 할 때 최고의 쾌감을 느낀다
- 어떤 모임은 재미있고 어떤 모임은 재미없다 — 자신의 설득력을 시험할 수 있는 모임만 재미있었던 것

### 관련 프레임워크
- McClelland: **Power (institutional)**
- CliftonStrengths: **Command, Woo, Communication, Activator**
- Fascination: **Power, Prestige**
- Working Genius: **Galvanizing**
- Wealth Dynamics: **Star, Deal Maker**

### 이 슈퍼파워가 만드는 직업들

| 직업 | 왜 이 직업인가 | 설득이 작동하는 방식 |
|------|---------------|---------------------|
| BD (사업개발) | 파트너를 설득하는 것이 본업 | 매 딜이 새로운 설득 실험 |
| 기획자/전략가 | 경영진을 설득하는 것이 본업 | 논리를 만들고 의사결정자를 움직인다 |
| 강연자/교육자 | 청중의 마음을 바꾸는 것이 본업 | 매 강연이 설득의 무대 |
| 변호사 | 판사·배심원을 설득하는 것이 본업 | 논리 + 증거 + 수사 |
| 로비스트/정책 기획 | 정책 결정자를 설득 | 사회적 규모의 설득 |
| 마케터 (전략) | 시장을 설득 | 대중의 인식을 바꾸는 게임 |

### 성공 원인

> **설득형이 성공하려면, "이기는 것"에서 "변화시키는 것"으로 목표를 바꿔야 한다.**

설득형의 함정은 **승리 중독**이다. 논쟁에서 이기는 것과 상대를 진정으로 변화시키는 것은 다르다. 이기기만 하면 적을 만들고, 변화시키면 동맹을 만든다. 성공하는 설득형은:

1. **듣기를 먼저 한다** — 상대의 프레임을 완벽히 이해한 뒤에 자기 논리를 꺼낸다. "당신 말이 맞아, 그리고..."가 "당신이 틀렸어"보다 100배 강력하다
2. **논리를 "스토리"로 포장한다** — 데이터와 논리는 머리를 움직이고, 스토리는 마음을 움직인다. 대성하는 설득형은 둘 다 쓴다
3. **설득의 결과를 축적한다** — 매 설득이 실험이라면, 실험 결과를 기록해야 한다. 어떤 프레임이 먹혔고, 어떤 것이 안 먹혔는지. 이 축적이 10년 뒤 통찰이 된다

### 흔한 오인
- **"매너가 좋고 세일즈가 천직"**이라고 생각 → 사실은 세일즈가 아니라 설득이 천직. 세일즈 안에서도 BD/기획/전략 쪽에서 더 빛난다
- **"네트워킹을 잘한다"**고 생각 → 네트워킹은 수단, 설득의 실험장이 목적. 관계형과 구분해야

### 멀티커리어 패턴
설득형은 **영향력의 범위를 넓혀간다**. 1:1 설득(영업) → 소규모 설득(팀 리딩) → 대규모 설득(강연/저술) → 사회적 설득(정책/미디어). 직업이 바뀌는 게 아니라 무대가 커지는 것이다.

## 4. 관계 (Relator)

### 핵심 질문
**"이 사람이 실망하지 않게 하려면?"**

### 무의식적 행동 패턴
- 관계 유지에 혼신의 힘을 쏟는다
- 연락이 끊기는 걸 견디지 못한다
- 누군가의 표정이 어두우면 즉각 감지한다
- 자기 이익보다 상대의 기분을 먼저 생각한다
- 한번 맺은 인연을 수십 년 유지한다

### 관련 프레임워크
- McClelland: **Affiliation (nAff)**
- SDT: **Relatedness**
- CliftonStrengths: **Relator, Empathy, Harmony, Individualization**
- Working Genius: **Enablement**
- StandOut: **Provider**
- Fascination: **Trust**

### 이 슈퍼파워가 만드는 직업들

| 직업 | 왜 이 직업인가 | 관계가 작동하는 방식 |
|------|---------------|---------------------|
| 고액 B2B 세일즈 | 고객과의 장기 신뢰가 핵심 | 한 고객을 10년 관리 |
| PB (프라이빗 뱅커) | 자산가의 삶 전체를 돌본다 | 돈이 아니라 사람을 관리 |
| 고액 부동산 중개 | 인생의 큰 결정에 동행 | 거래가 아니라 관계 |
| HR/인사 담당자 | 조직 내 사람의 접점 | 채용·성장·이탈 전 과정에 관여 |
| 간호사/상담사 | 돌봄이 직업 | 눈앞의 사람을 실망시키지 않는 것이 본업 |
| 커뮤니티 매니저 | 관계의 네트워크를 운영 | 사람과 사람을 잇는다 |

### 성공 원인

> **관계형이 성공하려면, "좋은 사람"에서 "없어서는 안 될 사람"이 되어야 한다.**

관계형의 함정은 **소진(번아웃)**이다. 모든 관계에 혼신을 쏟다 보면 자기가 비어간다. 그리고 "착한 사람"으로 남으면 대체 가능하다. 성공하는 관계형은:

1. **도메인 전문성을 하나 갖는다** — 관계 능력은 이미 있다. 여기에 부동산·자산관리·특정 산업 지식을 더하면 "좋은 사람"이 "없으면 안 되는 사람"이 된다
2. **관계에 구조를 만든다** — CRM이든 노트든, 관계를 기억에만 의존하지 않는다. 100명의 핵심 관계를 체계적으로 관리
3. **"No"를 말하는 연습을 한다** — 모든 관계에 동일한 에너지를 줄 수 없다. 핵심 관계를 선별하고, 나머지에는 건강한 거리를 둔다

### 흔한 오인
- **"사람을 좋아한다"**고 포괄적으로 정의 → 실제로는 "이 사람이 실망하지 않게" 하는 데 집중. 사교적인 것과 다르다
- **"네트워킹을 잘한다"**고 생각 → 설득형의 네트워킹(영향력 확장)과 달리, 관계형의 네트워킹은 유대 자체가 목적

### 멀티커리어 패턴
관계형은 **한 분야에 오래 머무는** 경우가 많다. 관계는 시간이 자산이기 때문이다. 다만 분야를 옮기더라도, 이전 관계가 새 분야의 자산이 된다. 그래서 관계형의 전직은 "리셋"이 아니라 "확장"이다.

## 5. 성취 (Achiever)

### 핵심 질문
**"다음 목표는 뭐지?"**

### 무의식적 행동 패턴
- 목표가 없으면 불안하다
- 달성하면 기쁨은 잠깐이고, 바로 다음 목표를 세운다
- 숫자로 측정 가능한 것을 좋아한다
- 경쟁 상대가 있으면 에너지가 올라간다
- 할 일 목록의 체크 표시가 하루의 보상

### 관련 프레임워크
- McClelland: **Achievement (nAch)**
- CliftonStrengths: **Achiever, Competition, Maximizer, Focus**
- Working Genius: **Tenacity**
- Kolbe: **Fact Finder 高**
- Wealth Dynamics: **Trader, Accumulator**

### 이 슈퍼파워가 만드는 직업들

| 직업 | 왜 이 직업인가 | 성취가 작동하는 방식 |
|------|---------------|---------------------|
| 영업 (목표 관리형) | 매달 명확한 숫자 목표 | 달성 → 초과 → 다음 분기 |
| 운동선수/코치 | 기록 갱신이 삶 | 0.01초를 줄이는 집착 |
| 경영자/CEO | 기업 성장이 성취 대상 | 매출·이익·시장점유율 |
| 트레이더 | 수익률이 점수판 | 매일 승패가 갈린다 |
| 프로젝트 매니저 | 마감·예산·품질 목표 관리 | 완료의 쾌감 |
| 시험 전문가 (CPA, 변호사 등) | 합격이라는 명확한 목표 | 시험 = 측정 가능한 성취 |

### 성공 원인

> **성취형이 성공하려면, "무엇을 달성할 것인가"의 방향을 잘 골라야 한다.**

성취형의 함정은 **목표 자체에 대한 무비판적 집착**이다. 잘못된 산을 오르면 정상에 도달해도 허무하다. 성공하는 성취형은:

1. **10년 단위의 방향을 먼저 정한다** — 단기 목표에 중독되면 방향 없이 달리게 된다. "이 목표를 달성하면 내 10년이 어떻게 달라지나?"를 묻는 습관
2. **팀의 성취로 확장한다** — 개인 성취에만 집중하면 리더가 될 수 없다. "내가 아니라 팀이 달성하게 하는 것"이 다음 레벨
3. **달성 후 "멈춤"의 시간을 갖는다** — 달성 → 즉시 다음 목표가 아니라, 달성 → 의미 부여 → 방향 점검 → 다음 목표. 이 "멈춤"이 방향을 교정

### 흔한 오인
- **"부지런하다"**고 자신을 정의 → 부지런함은 결과. 내적 동기는 "달성"이라는 쾌감
- **"완벽주의자"**라고 생각 → 성취형의 완벽주의는 목표 달성에 대한 것이지, 디테일 자체에 대한 것이 아니다 (후자는 수호형)

### 멀티커리어 패턴
성취형은 **목표의 크기를 키워간다**. 개인 목표 → 팀 목표 → 조직 목표 → 산업 목표. 직업이 바뀌기보다 목표의 스케일이 변한다. "작은 연못의 큰 물고기"에서 "큰 바다의 큰 물고기"로.

## 6. 수호 (Guardian)

### 핵심 질문
**"이게 무너지지 않게 하려면?"**

### 무의식적 행동 패턴
- 위험을 미리 감지하고 대비한다
- "만약 ~하면 어쩌지?"가 자동으로 떠오른다
- 질서와 체계가 무너지는 걸 견디지 못한다
- 디테일에서 오류를 찾아내는 능력이 뛰어나다
- 남들이 "괜찮을 거야"라고 넘길 때 혼자 대비한다

### 관련 프레임워크
- Reiss: **Order, Tranquility**
- CliftonStrengths: **Responsibility, Deliberative, Consistency, Discipline**
- Fascination: **Trust, Alert**
- Working Genius: **Tenacity** (지키며 완수하는 측면)
- Kolbe: **Follow Thru 高, Quick Start 低**

### 이 슈퍼파워가 만드는 직업들

| 직업 | 왜 이 직업인가 | 수호가 작동하는 방식 |
|------|---------------|---------------------|
| 운영 총괄 (COO) | 조직이 돌아가게 하는 사람 | 시스템이 무너지지 않게 |
| 법무/컴플라이언스 | 규칙을 지키게 하는 사람 | 위험을 사전에 차단 |
| QA/테스터 | 오류를 찾는 것이 본업 | 출시 전 결함 제거 |
| 재무/회계 | 숫자가 맞는지 확인하는 사람 | 1원도 안 맞으면 잠이 안 온다 |
| 보안 전문가 | 시스템을 지키는 사람 | 공격 시나리오를 미리 상상 |
| 에디터/교정자 | 글의 오류를 잡는 사람 | 디테일의 수호자 |

### 성공 원인

> **수호형이 성공하려면, "지키는 것"의 가치를 증명하는 언어를 배워야 한다.**

수호형의 함정은 **보이지 않는 공헌**이다. 문제가 안 생기는 건 수호형 덕분인데, 문제가 안 생기면 아무도 모른다. 성공하는 수호형은:

1. **"막은 것"을 가시화한다** — "이번 분기 컴플라이언스 위반 0건"이 아니라 "이 조치로 예상 손실 5억을 방지"로 말한다
2. **변화를 거부하는 게 아니라 "안전한 변화"를 설계한다** — 수호형은 변화에 저항하는 사람으로 오해받기 쉽다. "이렇게 하면 안전하게 변화할 수 있습니다"로 포지셔닝
3. **전문 영역을 하나 가진다** — 범용적 "꼼꼼한 사람"이 아니라 "사이버 보안의 전문가", "금융 규제의 전문가"로. 전문성 위에 수호 본능이 올라가면 대체 불가능

### 흔한 오인
- **"꼼꼼하다"**고 자신을 정의 → 꼼꼼함은 행동, 내적 동기는 "무너지지 않게"라는 수호 본능
- **"보수적이다"**라는 평가를 받음 → 보수적인 게 아니라 리스크를 미리 보는 것. 이것은 약점이 아니라 슈퍼파워

### 멀티커리어 패턴
수호형은 **같은 조직에 오래 머무는** 경우가 많다. 지키려면 그 대상을 깊이 알아야 하기 때문. 다만 전문 영역이 생기면, 그 전문성을 다른 조직에 이식하는 형태로 이동한다 (예: A사 법무 → B사 법무 → 법무법인 파트너).

## 7. 의미 (Idealist)

### 핵심 질문
**"이것이 세상에 어떤 의미가 있는가?"**

### 무의식적 행동 패턴
- 대의가 없는 일에서는 에너지가 0이 된다
- "그래서 이게 왜 중요한데?"를 끊임없이 묻는다
- 돈보다 사명감이 직업 선택의 기준
- 불의를 보면 행동하지 않고는 못 배긴다
- 자기 일의 사회적 의미를 늘 설명하고 싶어한다

### 관련 프레임워크
- Reiss: **Idealism, Honor**
- Daniel Pink: **Purpose**
- CliftonStrengths: **Belief, Significance**
- VIA: **Perspective, Integrity, Fairness**
- StandOut: **Equalizer, Pioneer**

### 이 슈퍼파워가 만드는 직업들

| 직업 | 왜 이 직업인가 | 의미가 작동하는 방식 |
|------|---------------|---------------------|
| 비영리 리더 | 사회적 미션이 직업 | 미션 = 존재 이유 |
| 교육자 | 다음 세대를 변화시키는 일 | 한 사람의 성장이 세상의 변화 |
| 사회적 기업가 | 돈과 의미를 동시에 | 비즈니스가 의미의 도구 |
| 저널리스트/다큐 감독 | 세상의 진실을 드러내는 일 | 보도 = 사회적 사명 |
| 정책 기획자 | 제도로 세상을 바꾸는 일 | 정책 = 대규모 변화의 도구 |
| ESG/지속가능성 담당 | 기업의 사회적 책임 | 이익과 의미의 교차점 |
| 의사/공중보건 전문가 | 사람의 생명을 다루는 일 | 가장 직접적인 "의미 있는 일" |

### 성공 원인

> **의미형이 성공하려면, "의미"와 "실행력"을 결합해야 한다.**

의미형의 함정은 **순수성의 덫**이다. "돈을 벌면 순수하지 않다", "타협하면 원칙이 무너진다"는 생각이 실행을 막는다. 성공하는 의미형은:

1. **비즈니스 모델을 배운다** — 의미만으로는 지속 불가능. "이 의미를 어떻게 돈으로 바꿀까"가 아니라 "이 의미를 유지하려면 어떤 수익 구조가 필요한가"
2. **작은 승리부터 만든다** — 세상을 한번에 바꾸려 하면 좌절. 한 사람, 한 커뮤니티, 한 도시부터. 작은 성공이 큰 변화의 증거
3. **같은 의미를 공유하는 동맹을 모은다** — 혼자 하면 순교자가 되고, 함께 하면 운동이 된다

### 흔한 오인
- **"착하다"**고 평가받음 → 착함이 아니라 정의감. 불의에 대한 분노가 동기
- **"이상주의자"**라고 평가받음 → 이상이 아니라 방향. "이렇게 되어야 한다"는 확신이 행동을 만든다

### 멀티커리어 패턴
의미형은 **의미를 중심으로 직업을 바꾼다**. 같은 사회 문제를 다른 각도에서 접근하는 형태. 예: 환경 문제 → 환경 NGO → 환경 스타트업 → 환경 정책 자문. 주제는 같고 도구가 바뀐다.

## 8. 감별 (Discerner)

### 핵심 질문
**"이 사람의 진짜 잘하는 게 뭐지?"**

### 무의식적 행동 패턴
- 남의 재능을 발견하면 경이로워한다
- 팀을 구성할 때 "누가 어디에 맞을까"를 자동으로 계산한다
- 사람들의 강점과 약점을 직관적으로 파악한다
- 쉬어야 할 때도, 화가 날 때도, 사람의 재능에 대해 생각한다
- "이 사람과 저 사람이 만나면 시너지가 날 텐데"가 입버릇

### 관련 프레임워크
- Working Genius: **Discernment**
- CliftonStrengths: **Individualization, Developer, Maximizer**
- StandOut: **Connector, Teacher**
- Wealth Dynamics: **Supporter** (재능을 모아 시너지를 만드는 측면)
- VIA: **Social Intelligence**

### 이 슈퍼파워가 만드는 직업들

| 직업 | 왜 이 직업인가 | 감별이 작동하는 방식 |
|------|---------------|---------------------|
| 벤처 투자자 (VC) | 창업자의 재능을 알아보는 것이 본업 | 사람에게 베팅 |
| 팀빌딩 전문가/코치 | 사람을 제 자리에 놓는 것이 본업 | 적재적소 설계 |
| 헤드헌터/인재 배치 | 사람과 자리를 매칭 | "이 사람이 저기 가면 빛나겠다" |
| 리더/경영자 (사람 중심형) | 팀의 역량을 극대화 | 개인 강점을 팀 성과로 전환 |
| 프로듀서/캐스팅 디렉터 | 누가 어떤 역할에 맞는지 판단 | 조합의 예술 |
| 멘토/인생 코치 | 한 사람의 슈퍼파워를 발견해준다 | "당신의 진짜 재능은 이거예요" |
| 교육 설계자 | 학생의 잠재력을 알아보고 경로 설계 | 개인별 성장 경로 |

### 성공 원인

> **감별형이 성공하려면, "알아보는 눈"에 "실행 시스템"을 결합해야 한다.**

감별형의 함정은 **감상에 머무는 것**이다. "이 사람 대단해"라고 감탄만 하면 관객이다. 성공하는 감별형은:

1. **판을 만든다** — 재능 있는 사람들이 만날 수 있는 장(場)을 설계한다. 이것이 커뮤니티든, 회사든, 프로젝트든
2. **언어를 만든다** — "당신의 재능은 이겁니다"를 말해줄 수 있는 프레임워크를 갖춘다. 직관을 구조화된 언어로 전환
3. **신뢰를 축적한다** — 감별형의 자산은 "저 사람이 알아본 건 맞았어"라는 트랙 레코드. 10번 알아보고 8번 맞으면, 그 다음부터는 한마디의 추천이 큰 힘을 가진다

### 흔한 오인
- **"사람을 좋아한다"**고 생각 → 관계형과 혼동. 관계형은 유대 자체가 목적이고, 감별형은 재능 발견이 목적. 감별형은 재능이 없는 관계에는 흥미를 덜 느낄 수 있다
- **"리더십이 있다"**고 평가받음 → 리더십은 결과. 내적 동기는 사람의 재능에 대한 경이와 호기심

### 멀티커리어 패턴
감별형은 **감별 대상의 규모를 키워간다**. 한 사람 → 한 팀 → 한 조직 → 한 산업의 인재 생태계. 나중에는 "이 분야의 킹메이커" 혹은 "사람을 보는 눈이 있는 어르신"이 된다.

## 슈퍼파워가 겹칠 때 — 멀티커리어의 원리

사람은 하나의 슈퍼파워만 가지지 않는다. 보통 **주(主) 슈퍼파워 1개 + 보조 슈퍼파워 1~2개**의 조합으로 작동한다. 이 조합이 멀티커리어를 만든다.

| 조합 | 생기는 멀티커리어 | 구체적 예시 |
|------|-------------------|------------|
| 탐구 + 창조 | 연구자-창업가 | 학계에서 발견한 것을 스타트업으로 사업화 |
| 탐구 + 설득 | 전략가-강연자 | 산업을 분석하고, 그 인사이트를 전파 |
| 창조 + 성취 | 연쇄 창업가 | 만들고 → 성장시키고 → 엑싯 → 다시 만들고 |
| 설득 + 관계 | 최고 영업인 | 설득력 + 관계 유지의 결합. B2B의 왕 |
| 설득 + 의미 | 사회운동 리더 | 대의를 위해 대중을 설득 |
| 관계 + 수호 | 최고의 RM | 관계를 지키면서, 그 관계를 안전하게 유지 |
| 관계 + 감별 | 인재 에이전트 | 사람의 재능을 알아보고, 관계로 연결 |
| 성취 + 수호 | 운영 전문가-경영자 | 목표를 세우고, 시스템이 무너지지 않게 달성 |
| 의미 + 창조 | 사회적 기업가 | 의미 있는 것을 새로 만든다 |
| 감별 + 설득 | 코칭 전문가-BD | 사람의 재능을 알아보고, 그 조합을 설득으로 실현 |
| 감별 + 탐구 | 인재 연구자-컨설턴트 | 사람의 동기 구조를 학문적으로 탐구 |

## 성공의 공통 원인 — 여덟 가지를 관통하는 세 가지

### 1. 자기 슈퍼파워를 정확히 아는 것

대부분의 사람은 자신의 강점을 **결과로 정의**한다 (머리가 좋다, 부지런하다, 꼼꼼하다). 그러나 결과는 동기가 아니다. 성공하는 사람은 결과 아래의 **동기를 정확히 안다**.

진단 방법:
- **쉬어야 할 때 무엇을 하는가?** — 그것이 당신의 슈퍼파워
- **화가 났을 때도 멈추지 않는 것은?** — 그것이 당신의 슈퍼파워
- **누가 시키지 않아도 하는 것은?** — 그것이 당신의 슈퍼파워
- **어떤 모임이 재미있고, 어떤 모임이 재미없는가?** — 재미있는 모임에서 당신의 슈퍼파워가 작동하고 있었다

### 2. 약점을 메꾸기보다, 강점 위에 도메인 전문성을 쌓는 것

관계형에게 "좀 더 논리적으로" 하라고 가르치는 것은 비효율적이다. 관계형에게 부동산 지식을 가르치는 것이 100배 효과적이다. **슈퍼파워 + 도메인 지식 = 대체 불가능한 사람**.

### 3. 자기 슈퍼파워에 맞는 직업을 고르는 것

같은 "영업" 직업 안에서도:
- **설득형**은 신규 고객 개척(BD)에서 빛나고
- **관계형**은 기존 고객 관리(AM/RM)에서 빛나고
- **성취형**은 목표 초과 달성(KPI 중심 세일즈)에서 빛난다

직업이 아니라, 직업 안의 **어떤 역할**에서 자기 슈퍼파워가 작동하는지를 아는 것이 진짜 커리어 설계다.

## 흔한 오인 패턴 종합

| 본인이 생각하는 강점 | 실제 슈퍼파워 | 왜 헷갈리나 |
|---------------------|-------------|------------|
| 머리가 좋다, 학벌이 무기 | 탐구 — 호기심이 연료 | 결과(학벌)를 원인(호기심)으로 착각 |
| 아이디어가 많다 | 창조 — 만들려는 충동 | 아이디어(원료)와 창조(행동)를 혼동 |
| 매너 좋고 세일즈가 천직 | 설득 — 논리로 마음을 바꾸는 실험 | 수단(세일즈)을 동기(설득)로 착각 |
| 사람을 좋아한다 | 관계 or 감별 | 관계 자체 vs 재능 발견. 같은 "사람 좋아함"이지만 동기가 다르다 |
| 부지런하다 | 성취 — 달성의 쾌감 | 행동(부지런)을 동기(성취)로 착각 |
| 꼼꼼하고 실수가 없다 | 수호 — 무너지지 않게 하는 집착 | 능력(꼼꼼)을 동기(수호)로 착각 |
| 착하다 | 의미 — 정의감 | 성품(착함)을 동기(의미 추구)로 착각 |
| 리더십이 있다 | 감별 — 재능에 대한 경이 | 결과(리더십)를 동기(감별)로 착각 |

## 마무리 — 슈퍼파워 컨설팅이라는 직업

이 프레임워크의 출발점은 하나의 관찰이었다:

> *"남들의 멋진 능력을 알아보고 경이하는데 너무 집착한다. 쉬어야 하는 순간에도, 다른 일을 해야하는 순간에도, 화가 나는 순간에도 나는 늘 그런 것들을 생각하고 살아간다."*

이것은 8번 감별(Discerner)의 전형적인 행동 패턴이다. 그리고 이 문서 자체가 — 8가지 슈퍼파워를 분류하고, 각각이 어떤 직업으로 발현되며, 어떻게 성공하는지를 정리하는 행위가 — 감별형 슈퍼파워의 작동이다.

**슈퍼파워 컨설팅**: 한 사람의 행동 패턴에서 내적 동기를 역추적하고, 그 동기에 맞는 직업·역할·환경을 설계해주는 일. 이것이야말로 감별 + 탐구의 조합이 만드는 자연스러운 멀티커리어의 한 형태다.
